Data Image Delivery

Small plugin to deal with responsive image delivery. Think responsive.io minus the cdn.

  • Usage

    $('img').dataImg();

  • Usage with options

    Small, Medium and Large breakpoints. Option for window resize.

    $('img').dataImg({
  sml: 400,
  med: 800,
  lrg: 1000,
  resize: true
});
